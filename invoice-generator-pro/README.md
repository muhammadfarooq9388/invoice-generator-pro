# 💼 Invoice Generator Pro

Invoice Generator Pro is a modern and responsive web application designed to create professional invoices quickly and efficiently. The application allows users to manage company and client information, add multiple products, calculate taxes automatically, generate invoice previews, print invoices, and export them as PDF files.

## 🚀 Features

* Company Information Management
* Client Information Management
* Auto Invoice Number Generation
* Invoice Date Selection
* Add Multiple Products
* Quantity & Price Calculation
* Automatic Tax Calculation (10%)
* Subtotal, Tax & Grand Total Summary
* Live Invoice Preview
* Download Invoice as PDF
* Print Invoice
* Local Storage Support
* Dark / Light Mode
* Responsive Design
* Toast Notifications

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* jsPDF Library
* Local Storage API

## 📖 How To Use

1. Enter company information.
2. Enter client information.
3. Add products with quantity and price.
4. Click "Add Product" to include items in the invoice.
5. Review the live invoice preview.
6. Verify subtotal, tax, and grand total calculations.
7. Download the invoice as a PDF file.
8. Print the invoice directly from the browser.
